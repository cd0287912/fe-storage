在 ts 中我们使用对象类型来描述对象。

对象类型可以是匿名

```ts
function createPerson(person: { name: string; age: number }) {}
```

可以使用接口

```ts
interface Person {
  name: string;
  age: number;
}
function createPerson(person: Person) {}
```

可以使用别名

```ts
type Person = {
  name: string;
  age: number;
};
function createPerson(person: Person) {}
```
